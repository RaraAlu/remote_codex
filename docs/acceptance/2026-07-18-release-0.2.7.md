## 遗留风险与结论

- 待补 Linux x64 本地 Controller、CJS Shim、官方扩展、任务创建和真实 Remote SSH E2E。
- 待按最低样本数采集启动、任务、固定远端探针和 MCP 延迟。
- 待实测设置恢复，以及停止、relay 断开、窗口关闭后的遗留进程数。
- Core 内置本地 Shell/文件工具的执行前硬阻断仍未实现，不能声明完整阶段 C 安全保证。
- 最终结论：**Windows x64 主链路限定范围通过；Linux x64 仅打包通过。**
