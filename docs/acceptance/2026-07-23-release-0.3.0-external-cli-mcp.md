## 遗留风险与结论

- 待补测：候选已安装，但必须由用户在活动 Remote SSH 窗口重载并验证官方 UI。
- 待补测：首次执行 `Codex Bridge: Enable Current CLI Conversation Control` 后，需要
  重启一次当前 CLI 会话，才能加载新 MCP 工具；之后由 Controller 自动刷新注册。
- 待补测：用真实 CLI 工具列出当前 VS Code 对话，读取 thread，介入活动 turn，并确认
  VS Code UI、`remote_exec` 和审计三者一致。
- 已知限制：当前 MCP 无法代替官方 UI 展示非 `full-access` 审批，遇到审批请求会拒绝。
- 最终结论：Linux x64 自动化候选通过，真实窗口与 Windows x64 尚未通过。
- 支持声明：不得从当前 Linux 构造结果声明 Windows 或真实 Remote SSH 已支持。
