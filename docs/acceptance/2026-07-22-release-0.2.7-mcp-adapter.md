## 遗留风险与结论

- 官方插件 Git watcher 仍在本地尝试监听远端 POSIX 根并重复记录 `ENOENT`；这属于
  `ROOT-PRIMARY` 能力缺口，不影响本次 MCP 与 `remote_exec` 结果。
- CodeGraph 工具名属于服务私有能力，升级时必须重跑 `tools/list`，不能沿用本次八工具
  结论。
- OpenSSH stdin 控制头只有自动化证据，真实回退链路待补测。
- Windows x64、普通本地窗口、关闭/断线/设置恢复和完整安全失败矩阵仍待补测。
- 最终结论：**Linux x64 Remote SSH 的通用 MCP 适配链路通过；完整发布硬门禁仍未
  通过。**
- 支持声明：本记录只声明
  `openai.chatgpt@26.715.61943` + 内置 `0.145.0-alpha.27` +
  Bridge `0.2.7` + Executor `0.2.6` 在 Linux x64 Controller 到 `g1_1` Remote SSH
  的本次 MCP 与 `remote_exec` 链路通过；不外推 Windows 或 OpenSSH 回退运行时。
