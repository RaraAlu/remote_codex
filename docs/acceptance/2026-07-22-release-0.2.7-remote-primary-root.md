## 能力探针

使用空白临时 `CODEX_HOME` 启动官方插件内置 app-server，不调用模型、不访问远端文件：

- `initialize.capabilities.experimentalApi=true` 后，`thread/start` 接受远程绝对路径
  `runtimeWorkspaceRoots`，同时接受本地控制目录 `cwd`。
- `thread/resume` 接受相同远程根并进入线程查找；临时线程没有 rollout，返回预期的
  `no rollout found`，不是路径或协议拒绝。
- `turn/start` 接受远程根、本地只读 sandbox 和逐轮应用上下文，并产生
  `turn/started`。
